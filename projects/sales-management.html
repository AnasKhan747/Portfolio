Sales Performance and Forecasting System

Client: ABC Company (UK based Leading IT Services Provider)
Role: Full-Stack Developer

ABC was using Autotask PSA but couldn't track LOB sales performance and future opportunities. We developed a custom web-based portal integrated with Autotask PSA to offer real-time data sync and advanced sales forecasting via dashboard, matrix view, and reporting tools.

Responsibilities

  • Developed full-stack web portal to integrate with Autotask PSA using ASP.NET and C#.
  • Created matrix-style dashboards for LOB sales tracking and contract renewal timelines.
  • Built hourly auto-import system for real-time data sync (CSV-based).
  • Implemented Calendar with events for Lead and Contract date using TUI Calendar API
  • Implemented roles and permission handling (Admin, Salesperson, Account Manager).
  • Designed and integrated dynamic reports: Sales by LOB, Profit Matrix, Lead Trends.
  • Collaborated with client team for gathering detailed LOB metrics and user flows.

Key Features Delivered

  • Dashboard: LOB breakdown by timeframe, expiring contracts, opportunity alerts.
  • Matrix View: Color-coded LOB status (sold, unsold, unavailable, etc.) with date toggles.
  • Diary Section: Monthly planner for follow-ups and contract renewals.
  • Profit Matrix for evaluating account manager performance and customer service mapping.
  • Detailed reporting: Lead stats, conversion trends, upselling analysis, and forecasting.
  • Role-based secure login system for Admins and Sales Teams.
  • Add/update contracts, upcoming LOBs, and opportunity statuses directly from the portal.

Tech Stack

ASP.NET C# .NET Framework Bootstrap 5 HTML JavaScript jQuery SQL Server CSV Parsing Google Charts

Client Impact

  • Improved visibility into sales team performance and LOB-specific trends.
  • Enabled sales forecasting and lead follow-ups, resulting in increased up-selling.
  • Eliminated reliance on multiple systems with unified web-based platform.
  • Streamlined contract renewal and follow-up process via diary and alert system.
  • Enhanced decision-making with matrix analytics and custom reports.
  • Reduced manual operations by implementing hourly data imports from Autotask PSA.
