Gas Cylinder Refilling and Tracking System

Client: XYZ Company (UK based Leading Refrigeration & Air Conditioning Distributor)
Role: Full-Stack Developer

XYZ Company faced critical issues in tracking gas cylinders, managing transactions, coordinating between branches, engineers, and customers, and ensuring timely collection and refilling. We designed and implemented a centralized digital system to modernize their operations across Admin, Customer, and Engineer levels.

Responsibilities

  • Led full-stack development of the Admin and Customer portals using DotNetNuke (DNN 9.10.02) with custom module development in C# and ASP.NET.
  • Built secure REST APIs and handled data transactions across Admin, Client, and Engineer interfaces.
  • Developed responsive UI using HTML, CSS, Bootstrap 5, JavaScript, jQuery.
  • Integrated Google Charts for dynamic visual analytics (bar/pie charts for branch-wise cylinder stats).
  • Implemented role-based access control (RBAC) for Admin, Branch Staff, and Customers.
  • Built features for CSV auto-import, cylinder scanning, and transaction logging (Reclaim/Transfer).
  • Implemented calendar with events for renewal and return date using TUI Calendar API
  • Coordinated with the client for requirement gathering, UAT, and deployment.
  • Implemented Google Maps API for tracking of location for cylinder pickup or return

Key Features Delivered

  • Admin Portal: Dashboard, cylinder tracking, diary/calendar for dues, gas type filters, hazardous/non-hazardous gas handling, auto CSV imports.
  • Customer Portal: Equipment management, user/engineer management, waste cylinder tracking, allocation, and transaction history.
  • Mobile App (for Engineers): QR scanning, gas/equipment transaction logs, real-time syncing with the KoolTrack portal.

Tech Stack

DotNetNuke (DNN) 9.10.02 C# .NET Framework SQL Server HTML CSS Bootstrap JavaScript jQuery Google Maps API Google Charts

Client Impact

  • Achieved 100% cylinder tracking accuracy across branches.
  • Reduced manual effort with automated gas transactions and collection records.
  • Improved customer satisfaction through request tracking and timely pickups.
  • Enabled 360-degree operational visibility for all stakeholders.
  • Strengthened environmental safety compliance with hazardous gas tracking.
  • Streamlined engineer service logging via the mobile app.
