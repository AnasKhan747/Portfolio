Team Logger

Client: Company Product
Role: Full-Stack Developer

A robust cross-platform desktop application for real-time employee monitoring, session tracking, productivity analytics, and live screen streaming.

Responsibilities

  • Architected the Electron.js application with modular, maintainable structure.
  • Integrated keyboard/mouse activity tracking and webcam capture logic using native modules.
  • Implemented screenshot capturing, live screen streaming, and role-based access controls.
  • Built secure RESTful APIs and integrated JWT authentication with encrypted token storage.
  • Developed offline support, auto-login, and session syncing using electron-store and background services.
  • Implemented security checks to prevent time manipulation using internet time and system audits.

Key Features Delivered

  • Cross-platform app compatibility (Windows, macOS, Linux).
  • Live screen streaming and periodic screenshots with lightweight performance impact.
  • Activity analytics, idle time detection, and productivity reporting via Google Charts.
  • Offline-first architecture with automatic data sync on reconnect.
  • Centralized policy enforcement for login, session start, and monitoring configurations.

Tech Stack

Electron.js Node.js MySQL JWT WebSocket uiohook-napi bcrypt day.js electron-store electron-builder sharp Axios Google Charts Bootstrap PHP Azure Git

Impact & Outcomes

  • Enhanced productivity transparency for remote and hybrid teams.
  • Reduced time theft and manual reporting errors.
  • Improved compliance with work-hour policies and security protocols.
  • Enabled scalable tracking for small teams to large enterprises.
