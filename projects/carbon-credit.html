Carbon Credits Exchange Portal

Client: Environmental Sustainability Initiative
Role: Full-Stack Developer

Developed an e-commerce platform to facilitate carbon credit transactions for companies, organizations, and individuals. The system supports browsing, purchasing, and managing certified carbon units.

Responsibilities

  • Developed the core platform to enable secure purchase and tracking of carbon credits.
  • Integrated Google Maps API to display company locations during transactions.
  • Used Google Charts API for data visualization and carbon offset impact metrics.
  • Designed and implemented templates using Liquid in PowerApps environment.

Key Features Delivered

  • User-friendly portal for browsing, selecting, and purchasing carbon credit units.
  • Live location maps for organizations purchasing credits using Google Maps API.
  • Dynamic charts representing carbon offset statistics and purchase history.
  • Seamless integration with PowerApps for scalability and workflow automation.

Tech Stack

PowerApps Liquid Template Google Maps API Google Charts API

Impact & Outcomes

  • Encouraged eco-conscious purchases and awareness through carbon offsetting.
  • Provided transparency and traceability with geolocation and visual data.
  • Enabled environmental reporting and certification based on verified transactions.
  • Contributed to digitization of sustainability goals for organizations.
