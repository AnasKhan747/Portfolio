Weekly Timesheet and Approval System

Client: US based Cleaning Services
Role: Full-Stack Developer

Designed and developed a secure weekly timesheet and approval system that enables employees to log their working hours, manage time zones, submit leave requests, and receive customizable reminders. The system includes timesheet approval workflows and detailed reporting features.

Responsibilities

  • Developed timesheet entry and submission modules using ASP.NET and DNN with a Bootstrap frontend.
  • Implemented time zone handling using day.js to support global teams.
  • Created dashboard and charts using Google Charts API
  • Created leave request and approval components.
  • Integrated role-based dashboards and approval panels.
  • Built notification engine with configurable reminders via email and UI alerts.
  • Handled secure CRUD operations using C#, SQL Server, and jQuery AJAX.

Key Features Delivered

  • Weekly Timesheet Entry: Log hours per day, select project and task, and adjust entries based on time zone.
  • Leave Management: Apply and track time off with manager approvals.
  • Time Zone Support: Automatically adjust times based on user's local time using day.js.
  • Reminder Notifications: Custom rules to remind users before submission deadlines.
  • Approval Workflow: Managers can view, approve, reject, and comment on submitted timesheets.
  • Role Management: Roles such as Employee, Manager, and Admin with separate access levels.
  • Audit logs for submissions, changes, and approvals.

Tech Stack

DotNetNuke (DNN) C# .NET Framework SQL Server HTML CSS Bootstrap JavaScript jQuery day.js

Client Impact

  • Streamlined weekly timesheet submission and approval for remote and in-house teams.
  • Improved accuracy and accountability in work-hour tracking across time zones.
  • Reduced missed submissions through smart reminders and notification preferences.
  • Enabled better leave planning and visibility for team leads.
  • Increased manager efficiency with bulk review and approval tools.
  • Facilitated audit and reporting for HR and finance purposes.
